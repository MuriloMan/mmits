# MMITS - IT PARTNER 

## Init
> $npm install

## to build
> $./mmits-build

## to deploy
> $./mmits-deploy.sh

## i18n
> $static-i18n -o="dist/" -l="pt" -i="pt" app/

## Run
To run, open 2 terminals, the first one:
> $gulp

The second one:
> $http-server
